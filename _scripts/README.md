# 자체감사결과 스크래핑 재개 방법

`full_scrape.js`는 공공감사포털(pap.go.kr) 자체감사결과 목록을
2025-07-07 ~ 2026-07-07, 기관분류=공공기관 조건으로 전부 내려받는
스크립트입니다. `checkpoint.json`에 진행 상황(다음 페이지, 파일명
중복 처리 registry)이 저장되어 있어 중단 후에도 이어서 실행할 수
있습니다.

## 재개 방법

1. 이 폴더(`_scripts/`)를 작업 디렉터리로 사용할 임시 경로에 복사
   (checkpoint.json은 스크립트와 같은 폴더에 있어야 함)
2. 실행:
   ```
   node full_scrape.js /home/user/data/자체감사결과
   ```
3. `checkpoint.json`의 `nextPage`부터 자동으로 이어서 진행됨
4. 진행되며 생성되는 파일은 `/home/user/data/자체감사결과/`에 저장됨
5. `progress.log`에 전체 처리 로그가 계속 append됨

## 주의

- 프록시 포트가 세션마다 바뀔 수 있으므로 `lib.js`가 `$HTTPS_PROXY`
  환경변수에서 자동으로 읽어옴 (하드코딩 아님)
- 검증 규칙: 각 보고서의 모든 조치사항 첨부파일을 실제로 조회해서
  파일명+크기가 다르면 별도 문서로 취급, 같으면 1개로 합침. 파일명은
  `기관명_{연도}년 {감사분야}` 규칙이며 텍스트가 겹치면 등록일
  최신순으로 `(1)`, `(2)`... 번호를 붙임
